Butterknife

Standalone macOS 3D print assistant

Download

Butterknife is a standalone macOS app. Python is not required.

Download Butterknife

Setup

  1. Download and unzip the app
  2. Drag Butterknife.app into Applications
  3. Open the app
  4. If blocked: System Settings → Privacy & Security → Open Anyway

Butterknife

AI-Assisted 3D Print Slicer for macOS

What is Butterknife?

  • With Butterknife, anyone can generate a ready-to-go 3D printable file by simply describing their print. Instead of tinkering with hobbyist jargon on other slicers, use Butterknife to 3D print with plain English. Butterknife's intent engine understands your print and, with the help of SuperSlicer for gemeotrical calculations, generates an expertly optimized 3D print file (.gcode) in seconds.

    Setup Instructions

    1. Unzip the downloaded file.
    2. Drag Butterknife.app into your Applications folder.
    3. Download and install SuperSlicer if you don’t already have it (if downloading for the first time, permission might need to be granted via System Settings → Privacy & Security).
    4. Right-click Butterknife → Open (for first launch only, permission might need to be granted via System Settings → Privacy & Security).
    5. Locate SuperSlicer when prompted.
    6. Slice away!

Download

Download Butterknife v0.2.0 for macOS
macOS only • Intel & Apple Silicon supported

Setup Instructions

  1. Unzip the downloaded file.
  2. Drag Butterknife.app into your Applications folder.
  3. Download and install SuperSlicer if you don’t already have it.
  4. Right-click Butterknife → Open (first launch only, due to macOS security).
  5. Locate SuperSlicer when prompted.
  6. Start slicing your 3D models.
